# 📋 日次タスク管理アプリ

毎日のタスクを管理し、完了率をカレンダーで可視化できるPWA（Progressive Web App）です。

## ✨ 主な機能

### タスク管理
- ✅ 繰り返しタスクの作成と管理
- 📅 開始日・終了日の設定
- 📝 タスクごとのメモ機能
- ✏️ タスクの編集・削除

### 一時停止機能
- ⏸ タスクの一時停止（今日だけ / 今日と明日 / カスタム期間）
- 🔄 停止中のタスクを即座に復帰
- 📊 停止期間は進捗率から除外、履歴は保持

### 進捗管理
- 📈 日次完了率の表示
- 📆 月間カレンダーで進捗を可視化
- 🎯 完了率を色分け表示（80%以上：緑、50%以上：黄、50%未満：赤）

### データ管理
- 💾 ローカルストレージに自動保存
- 📤 データのエクスポート（JSON形式）
- 📥 データのインポート
- 📱 デバイス間でのデータ共有が可能

### PWA機能
- 📱 ホーム画面に追加可能（アプリのように使用）
- 🔌 完全オフライン対応
- ⚡ 高速起動

## 📱 使い方

### 1. アクセス方法

#### オプションA：GitHub Pages（推奨）
上記のデモURLにアクセス

#### オプションB：ローカルで実行
```bash
# リポジトリをクローン
git clone https://github.com/kuru77/task-tracker.git
cd task-tracker

# HTTPサーバーを起動（Python 3の場合）
python -m http.server 8000

# ブラウザで開く
open http://localhost:8000
```

### 2. ホーム画面に追加（iPhone/iPad）

1. **Safariで開く**（重要：Chromeなどでは制限あり）
2. 画面下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

これでアプリのようにホーム画面から起動できます！

### 3. 基本的な使い方

#### タスクの追加
1. 「今日のタスク」タブで、タスク名を入力
2. 開始日と終了日を設定（終了日は任意）
3. 必要に応じてメモを追加
4. 「追加」ボタンをクリック

#### タスクの完了
- タスクの横にあるチェックボックスをクリック

#### タスクの一時停止
1. タスクの「一時停止」ボタンをクリック
2. 停止期間を選択：
   - **今日だけ**：今日のみ停止、明日から自動復帰
   - **今日と明日**：2日間停止
   - **それ以上**：カレンダーから期間を指定
3. 「停止期間を追加」をクリック

#### 停止中のタスクを復帰
1. 「停止中」タブを開く
2. 復帰したいタスクの「今すぐ復帰」ボタンをクリック
3. 確認ダイアログで「OK」
   - 過去の停止記録は保持されます
   - 今日から通常のタスクとして復帰します

## 💾 データ管理

### デバイス間でデータを共有する方法

1. **デバイスAでデータをエクスポート**
   - 「設定」タブ → 「データをエクスポート」
   - JSONファイルがダウンロードされます

2. **デバイスBにデータをインポート**
   - JSONファイルをデバイスBに転送（メール、クラウドなど）
   - 「設定」タブ → 「データをインポート」
   - ファイルを選択

### バックアップの推奨頻度
- 週1回程度の定期バックアップをおすすめします
- ブラウザのデータを削除すると、タスクも消えます

## ❓ よくある質問

**Q: データが消えてしまいました**  
A: ブラウザのキャッシュやデータをクリアすると消えます。定期的なエクスポートをおすすめします。

**Q: 自動同期はできますか？**  
A: 現在はローカルストレージのみです。Firebaseなどのバックエンドを追加すれば可能です。

**Q: iPhoneでホーム画面に追加できません**  
A: Safariブラウザを使用してください。Chromeなどでは制限があります。

**Q: オフラインで使えますか？**  
A: はい、Service Workerにより完全にオフラインで動作します。
